**🚀 AI WhatsApp Sales Automation System**

An AI-powered sales automation platform that simulates intelligent WhatsApp customer conversations using NLP, Machine Learning, Lead Qualification, CRM Storage, and Real-Time Dashboard Analytics.

This project was developed to automate sales conversations, improve lead management, and reduce manual workload for sales teams.

-----------------------------------------------------------------------------------------------------------

**📌 Project Overview**

Many businesses struggle to manually handle large volumes of WhatsApp customer conversations. This project solves that problem by building an AI-driven automation workflow that can:

Automate customer replies
Detect customer intent
Extract lead information
Qualify and prioritize leads
Escalate important conversations
Store CRM records
Display analytics through a dashboard

The system works completely on local lightweight models without requiring paid APIs or cloud subscriptions.

-----------------------------------------------------------------------------------------------------------

**🎯 Key Features
✅ AI-Powered Intent Detection**

  Detects customer intent such as:
    1.Pricing Inquiry
    2.Demo Request
    3.Product Inquiry
    4.Support Request

-----------------------------------------------------------------------------------------------------------

**✅ Lead Information Extraction**

Automatically extracts:
  a.Budget
  b.Location
  c.Requirement
  d.Timeline/Urgency

-----------------------------------------------------------------------------------------------------------

**✅ Intelligent Lead Scoring**

Analyzes customer quality using:
  a.Budget
  b.Urgency
  c.Demo interest
  d.Business requirements

Lead categories:
  --HOT LEAD 🔥
  --MEDIUM LEAD 🟡
  --LOW LEAD 🟢

-----------------------------------------------------------------------------------------------------------

**✅ Escalation Detection**
Automatically flags high-priority leads for sales team attention.

**✅ CRM Database Integration**
Stores customer conversations and lead data using SQLite.

**✅ WhatsApp-Style Chat Interface**
Interactive Streamlit chat UI simulating WhatsApp conversations.

**✅ Real-Time Sales Dashboard**
Displays:
  Total Leads
  Hot Leads
  Escalated Cases
  CRM Records
  Lead Analytics

**⚙️ Installation**

**1️⃣ Clone Repository**
git clone YOUR_GITHUB_REPO_LINK

**2️⃣ Create Virtual Environment**
python -m venv venv

**3️⃣ Activate Virtual Environment**
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate

**4️⃣ Install Dependencies**
pip install -r requirements.txt

**5️⃣ Install spaCy Model**
python -m spacy download en_core_web_sm

**▶️ Running The Application**
Run Flask Backend
python app.py
Run Streamlit UI
streamlit run dashboard/main_app.py
